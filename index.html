About This App

連珠(五目並べ)の棋譜閲覧アプリです。.lib形式のファイルを読み込んで棋譜を閲覧、またダウンロード、画面下部からテキストでの棋譜貼り付けに対応しています。

製作者は連珠(五目並べ)関係のYouTubeチャンネルを運営しております。
応援していただけると嬉しいです。

youtube(那智暴虐のれんじゅいし【競技五目並べ】)

Select Encoding

ファイルの文字コードを選択してください。
(通常は Shift_JIS です)

Select Encoding (Save)

保存する文字コードを選択してください。
(lib内で使用されている言語に合わせてください)
選択された文字コードとlibの文字コードが一致しない場合、正確に保存されません。 br>Choose the character code. (Same language as you use in lib)
If the code you choose and the one used in lib are different, app can't save lib properly.
